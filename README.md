# Portfolio Website

A modern, responsive portfolio website built with React and Vite.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and visit the local URL shown in the terminal (usually http://localhost:5173)

## Customization

Edit the following files to personalize your portfolio:

- **src/components/Header.jsx** - Update your name in the logo
- **src/components/Hero.jsx** - Update your name, title, and description
- **src/components/About.jsx** - Add your bio and social links
- **src/components/Projects.jsx** - Add your projects with descriptions, technologies, and links
- **src/components/Skills.jsx** - List your skills and technologies
- **src/components/Contact.jsx** - Update your contact information

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder, ready to deploy to any static hosting service.
